# rexauto gabaritos

Per-binary **pre-discovered function cures** (the `functions.toml` overrides
rexauto's heal finds). Keyed by the SHA-256 of the title's `default.xex`, so a
cure set always matches the exact code image. When rexauto builds a title whose
`default.xex` hash matches a file here, it seeds these and **skips the slow
auto-cure cycle**. Add yours with `rexauto ... --publish-gabarito` and commit it.
